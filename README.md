# next-ts-template

#### 介绍

基于 `Next.js 14` 实现的一个项目模版，大佬的介绍文章可见：
[给上市公司从0到1搭建Next.js14项目](https://juejin.cn/post/7344571324305981503)。

中文开发文档：https://next.nodejs.cn/docs

#### 安装教程

`git clone` 项目到本地即可。

#### 使用说明

下载依赖：

```bash
npm install
```

开发启动：

```
npm run dev
```

生产打包：

```
npm run build
```

生产运行：

```
npm start
```
